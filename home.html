Heart Disease Survival Prediction

Project Aim

The main purpose of this project is to predict Heart Diseae survived or not? in these given situations


Input and output

Input: Passenger

Output: Prediction (Survived / Not Survived)


Representation of Input (Passenger)

  Input (Passenger) is Represented as 4 Input Attributes

  Input Attributes 01– age (Passenger Class)

    Possible Value 01 – Age

    Possible Value 02 – Age

    Possible Value 03 – Age


  Input Attributes 02– Sex

    Possible Value 01 – Male

    Possible Value 02 – Female


  Input Attributes 03– CP

    Possible Value 01 – One

    Possible Value 02 – Two

    Possible Value 03 – Three


  Input Attributes 04– Trestbps

    Possible Value 01 – 145

    Possible Value 02 – 130

    Possible Value 02 – 130


Representation of Output

   Output is Represented as a single Output Attributes

    Output Attribute – Survived

    Possible Value 01 – Yes

    Possible Value 02 – No
